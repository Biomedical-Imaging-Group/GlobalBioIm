List of Methods

\[\]

This section gives an overview of all the methods for each abstract class (and thus all inherited classes). All the core methods (i.e. with “_”) can be reimplemented in any derived class (if relevant). Otherwise the default implementation of the mother class is used.

Note the applyJacobianT() method of Map becomes sealed (i.e. not re-implementable in derived classes) from the level of LinOp and Cost. The reason is that for these classes, the Jacobian is respectively the adjoint and the gradient. Instead, the methods applyAdjoint() and applyGrad() are implemented.
